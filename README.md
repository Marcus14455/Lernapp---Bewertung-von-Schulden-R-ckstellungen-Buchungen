# Lerntrainer – Verbindlichkeiten & Rückstellungen

Lern-App für die Prüfungen (REWE):

- Fremdwährungsverbindlichkeiten (HGB & Steuerrecht)
- Rückstellungen (Begriff, Bewertung, Abzinsung, Buchungssätze)

**Nutzung:**
1. Link im Browser öffnen.
2. Erst mit „Karteikarten“ lernen.
3. Danach im „Quiz“-Modus testen.
